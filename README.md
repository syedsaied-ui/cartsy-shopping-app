# 🛒 Cartsy Shopping App

An e-commerce style shopping cart application built using React, with product data fetched from a backend API hosted on Railway.

## 🌐 Live Demo

🔗 https://cartsy-saied.vercel.app

## 🚀 Features

* Add and remove items from cart
* Manage cart state with React
* Dynamic price calculation
* Fetch products from backend API
* Responsive user interface

## 🛠️ Technologies Used

* React
* JavaScript (ES6+)
* CSS
* Node.js
* Express.js Backend (Hosted on Railway)
* REST API

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/syedsaied-ui/cartsy-shopping-app.git
```

### 2. Navigate to the project folder

```bash
cd 3-cartsy-shopping-app
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

## 🌐 Usage

After starting the development server, open:

```text
http://localhost:5173
```
