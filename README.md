# 🛒 Cartsy Shopping App

An e-commerce style shopping cart application built using React, with a separate backend server for handling product data.

## 🚀 Features

- Add and remove items from cart
- Manage cart state
- Dynamic price calculation
- Fetch products from backend server

## 🛠️ Technologies Used

- React
- JavaScript (ES6+)
- CSS
- Express.js (Prebuilt Backend)
- Node.js
- REST API

## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/syedsaied-ui/cartsy.git
```

### 2. Install dependencies (Frontend)

```bash
npm install
```

---

## ▶️ How to Run the Project

### 🔹 Step 1: Start Backend Server

1. Open a terminal
2. Navigate to backend folder:

```bash
cd 2-actual-backend
```

3. Install dependencies (if needed):

```bash
npm install
```

4. Start backend server:

```bash
npm start
```

👉 This will start the API server for products

---

### 🔹 Step 2: Start Frontend (React App)

1. Open a **second terminal**
2. Navigate to main project folder
3. Run:

```bash
npm run dev
```

---

## 🌐 Usage

After both servers are running:

- Frontend runs on:

```
http://localhost:5173
```

- Backend runs on:

```
http://localhost:8080
```

👉 Make sure **both are running**, otherwise products will not load.
